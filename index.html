Mod-Clash is a thing!

Games like ArmA 3 are great for modding, however sometimes having too many mods running concurrently can cause unexpected results

Where possible, try to minimise your mod-profile before submitting a report, as this helps narrow down the cause of any issues

How to report bugs

Any intel about breakages or problems is always gratefully received. Here are some tips to help us get to the root-cause as quickly as possible

Complete the form as best you can

We know it can be a pain, but honestly the more succinct detail you can provide, the better our chances are of understanding, tracking and then illiminating the offending code

Test against a bare-bones mod-list

I know we said it above, but we'll say it again, please ensure that (where possible) you report bugs that have been found when using *just C2 and CBA* - ruling out mod-clash can save a lot of time all round

Establsh if you are able to reproduce the issue

Being able to replicate an issue is *very helpful* when tracking bugs, so any intel on your steps to recreate an issue would be very welcome

Be as clear as you can in your report

We have provided a form (below) that we think enable sthe best chance of hunting down pesky bugs. While not all fields are mandatory, completing the form as best you can will really improve our changes of tracking down the error

Include a screenshot or video link

Using images to demonstrate an error or behaviour can be very useful, and ideally this should be (if possible) in a single image

Be patient

Like most mod projects, this is a hobby not a full time occupation, so your patience when resolving reported issues is greatly appreciated
